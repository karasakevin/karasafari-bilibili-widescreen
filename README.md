# B站自动宽屏（Edge 扩展）

在哔哩哔哩（Bilibili）网站播放普通视频时，自动点击一次“宽屏”按钮。

## 功能

- 自动触发宽屏（仅在哔哩哔哩域名生效）
- 避免反复切换：同一个视频只尝试一次

## 安装（Microsoft Edge）

1. 打开 `edge://extensions/`
2. 打开右上角 **开发人员模式**
3. 点击 **加载解压缩的扩展**
4. 选择本项目文件夹（包含 `manifest.json` 的目录）

## 开发

修改 `content-script.js` 后，到 `edge://extensions/` 点击本扩展的 **重新加载** 即可生效。
